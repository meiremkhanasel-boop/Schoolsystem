КАК ИСПОЛЬЗОВАТЬ:
═════════════════════════════════════════════════════════════════════════

1. Получить токен через вход
2. Сохранить токен в переменную
3. Использовать в заголовке: Authorization: Bearer <token>
4. Выполнять операции с данными

Пример (JavaScript):
  const token = await login('admin', '123');
  const students = await fetch('/api/students', {
    headers: { 'Authorization': `Bearer ${token}` }
  }).then(r => r.json());

РЕШЕНИЕ ПРОБЛЕМ:
═════════════════════════════════════════════════════════════════════════

Порт занят:
  Измените server.port в application.properties

401 Unauthorized:
  Проверьте токен в заголовке Authorization

404 Not Found:
  Проверьте URL (должен содержать /api/)

Приложение не запускается:
  Выполните: mvnw.cmd clean compile
